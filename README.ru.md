Fast Collage For Win это небольшое приложение на Python для быстрого создания коллажей.
Два режима работы

Свободный коллаж

Подходит, если вы хотите полный контроль:
* пустой холст без ограничений;
* неограниченное количество изображений (ограничение только по памяти);
* свободное перемещение, масштабирование и вращение фото;
* удобен для креативных и нестандартных композиций.

Случайный коллаж

Подходит, если нужно быстро получить аккуратный результат:
* вы выбираете количество фотографий;
* приложение автоматически генерирует шаблон под это количество;
* шаблон заполняет весь холст без пустых областей;
* каждое фото размещается в отдельной рамке (ячейке);
* изображения можно менять местами, масштабировать, поворачивать и отражать внутри рамок.

Работа с изображениями

* загрузка изображений:
  * перетаскиванием прямо из проводника;
  * через специальную панель превью;

* изображения можно:
  * масштабировать(ctrl+mouse);
  * вращать(shift+mouse);
  * зеркалить по горизонтали и вертикали(ctrl+shitf+H, ctrl+shift+V);
  * настраивать положение внутри рамки(Z+mouse);

* при работе с шаблонами:
  * всё, что выходит за границы рамки, автоматически скрывается;
  * изображения не мешают друг другу.

* масштабирование холста колесом мыши(ctrl+mouse);

* отмена и повтор действий(ctrl+Z, ctrl+Y);

Панель превью изображений

Приложение включает удобную боковую панель:
* сюда можно заранее загрузить все фотографии;
* отображаются миниатюры изображений;
* изображения можно перетаскивать на холст или в ячейки шаблона;
* использованное изображение можно убрать из коллажа и вернуть обратно в панель(клавиша X);
* панель можно скрыть или показать при необходимости.

Это избавляет от постоянного поиска файлов в папках и ускоряет работу.

Экспорт результата

* сохранение коллажа в популярных форматах (PNG, JPEG);
* экспорт в точном размере холста;
* готово для печати или публикации.

Кому подойдёт Fast Collage For Win
* пользователям, которым нужно быстро собрать красивый коллаж;
* дизайнерам и фотографам для подготовки макетов;
* тем, кто делает фотокниги, постеры, превью, сторис;
* всем, кто хочет удобный, понятный и гибкий инструмент без лишней сложности.
